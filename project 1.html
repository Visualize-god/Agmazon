Product Overview

Gradific is a web-based grading tool tailored to meet the grading needs of academic institutions. It empowers admins to track learner performance, enables moderators (teachers) to manage assignments and grades efficiently, and motivates learners with real-time feedback. Gradific’s primary focus is Nigeria.

Problem Statement

Academic institutions in Nigeria struggle with traditional grading methods which are stressful and time-consuming.

Goals & Solution

Goals:

  • Streamline assignment creation and submission workflows.
  • Reduce manual grading workload with automation.
  • Ensure fairness and clarity in grading criteria.

Solution:

A comprehensive platform with:

  • Task creation and assignment features.
  • Member onboarding and submission portals.
  • Manual and automated grading tools.

Market & User Research

Market Insights:

  • Nigeria's institutions have increasing demand for scalable grading tools.
  • Many institutions rely on manual processes or rigid LMSs with limited flexibility.

User Persona

1. The Owner (Admin)

Goals: Track performance, manage moderators, make data-driven decisions.

Pain: Cannot monitor thousands of learners in real time.

2. The Moderator (Instructor)

Goals: Grade faster, give feedback, manage tasks.

Pain: High volume of grading and limited tools for delegation.

3. The Member (Student)

Goals: Submit on time, get feedback, get promoted.

Pain: Unclear grading system and delayed results.

User Flow & Customer Journey Map

Key Flows:

  • Owner Onboarding → Workspace Creation → Moderator Invitation → Path Assignment.
  • Moderator Task Creation → Set grading criteria → Publish to assigned paths.
  • Member Submission → View dashboard → Submit link → Await feedback.
  • Grading → Manual or automated → Performance updated.

Customer Journey Highlights:

  • Moderators can assign tasks and grades within a unified dashboard.
  • Learners can be graded automatically and get feedback.
